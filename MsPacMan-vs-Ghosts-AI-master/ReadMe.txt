In order to run the project, PacMan-vs-Ghosts-Agents
have to have a reference to PacMan-vs-Ghosts module.
To do that (In "IntelliJ IDEA 15 CE"): 

1)Import PacMan-vs-Ghosts-Agents folder to IntelliJ

2)Click File-> Project Structure 

3)Select Modules from the left most panel

4) Click + icon at the bottom of right most panel and add PacMan-vs-Ghosts module


The simulator used in the project had been taken from the following repository:
https://github.com/kefik/MsPacMan-vs-Ghosts-AI
